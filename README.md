# Finanzas Pro

Una aplicación moderna y premium para la gestión de finanzas personales, construida con Vanilla JavaScript y CSS moderno.

## Características

- ✅ Diseño Premium con Glassmorphism
- 📱 Interfaz Responsiva (Desktop y Móvil)
- 🔒 Gestión de Sesión y Workspaces
- 📊 Dashboard con estadísticas en tiempo real
- 👤 Gestión de Beneficiarios y Categorías

## Instalación

1. Clona el repositorio
2. Ejecuta un servidor local:
   ```bash
   npx http-server -p 3001
   ```
3. Abre `http://localhost:3001` en tu navegador.

## Despliegue en GitHub Pages

Esta aplicación está lista para ser desplegada en GitHub Pages siguiendo estos pasos:

1. Sube tu código a GitHub (si aún no lo has hecho).
2. En tu repositorio, ve a **Settings** > **Pages**.
3. En la sección **Build and deployment**, selecciona la rama `main` y la carpeta `/ (root)`.
4. Haz clic en **Save**.
5. Tu aplicación estará disponible en `https://<tu-usuario>.github.io/FinanzasPrevio/` en unos minutos.

## Requerimientos Técnicos

- HTML5 / CSS3 (CSS Variables, Grid, Flexbox)
- JavaScript Moderno (ESM)
- API Fetch
